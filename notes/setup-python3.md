# Setup Python3

## Environment

- MacOS
- used HomeBrew

## Steps

1. xcode-select --install.
1. ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
1. brew doctor Your system is ready to brew.
1. brew install python3.
1. python3 --version Python 3.7.5.
